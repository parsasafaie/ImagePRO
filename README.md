## ImagePRO meanings
imagePRO have 2 meanings:

1. Image Processing
2. Image Professional

## Features
- Detects faces in images.
- Displays images with bounding boxes around detected faces.
- Writes bounding box coordinates of detected faces to a text file.

## Requirements
- Python 3.x
- torch
- facenet_pytorch
- Pillow
- matplotlib
- Opencv

## Installation
First, clone the repository:
```
git clone https://github.com/parsasafaie/ImagePRO.git
```
Then, in project folder, use this command for install dependencies.
```
pip install -r requirments.txt
```
